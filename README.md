# Function-Person
В данной программе реализован алгоритм работы с персоналом. 
Автоматизирован процесс считывания данных из файла и загрузка в файл с помощью сериализации класса.
В программе есть недочёты и не добавленные функции, которые будет реализованы в скором времени.
# Возможности
* Получение информации о сотрудниках
* Внести сотрудника в базу сотрудников
* Поиск сотрудников по имени, фамилии и возрасту
* Сортировка сотрудников
* Удаление сотрудника по id
* Удаление всех сотрудников
